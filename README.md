# Морской Бой

Многопользовательская браузерная игра в морской бой с ручной расстановкой флота, матчами между разными ПК и PHP/MySQL-бэкендом для обычного shared hosting.

## О проекте

Проект сделан как полноценная онлайн-игра:

- два игрока подключаются к одной комнате по коду
- каждый расставляет свой флот по правилам
- ходы синхронизируются через PHP API и polling
- матч работает на обычном CGI/PHP-хостинге без Node.js-сервера
- есть светлая и тёмная темы
- интерфейс адаптирован под ПК и мобильные устройства

## Основные возможности

- создание комнаты и вход по коду
- ручная расстановка флота с проверкой правил
- запрет касания кораблей даже по диагонали
- повторный ход после попадания или потопления
- уведомления об ошибках и игровых событиях
- сдача во время боя
- отмена комнаты на этапе расстановки
- экран финальной статистики
- реванш только по согласию двух игроков
- мобильное боковое меню в матче

## Правила флота

Поле: `10x10`

Состав флота:

- `1` корабль на `4` клетки
- `2` корабля на `3` клетки
- `3` корабля на `2` клетки
- `4` корабля на `1` клетку

Корабли не могут соприкасаться друг с другом ни сторонами, ни углами.

## Локальный запуск

Установить зависимости:

```bash
npm install
```

Запустить фронтенд в режиме разработки:

```bash
npm run dev
```

Собрать production-версию:

```bash
npm run build
```

Подготовить релиз для хостинга:

```bash
npm run release
```

## Релиз

Команда `npm run release`:

- собирает фронтенд
- готовит папку `release/morskoy-boy/`
- копирует туда нужные файлы для хостинга
- создаёт архив `release/morskoy-boy-hosting.zip`

Этот архив можно сразу загружать на хостинг.

## Структура проекта

```text
src/         React-клиент
api/         PHP API для комнат, расстановки, выстрелов и реванша
shared/      Общая игровая логика и типы
database/    SQL-схема базы данных
scripts/     Служебные скрипты, включая сборку релиза
release/     Готовые архивы и staging релиза
```
